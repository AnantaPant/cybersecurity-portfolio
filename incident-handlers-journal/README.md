# Incident Handler’s Journal – Ananta Pant

This journal documents my experience analyzing security incidents and understanding how professionals respond to cyber threats. I practiced reviewing alerts, identifying suspicious activity, and documenting findings clearly.

## Key Skills Demonstrated
- Understanding incident response steps  
- Reviewing logs and alerts  
- Identifying indicators of compromise  
- Documenting security events clearly  
- Learning how incidents are triaged and escalated  

## What I Learned
This assignment helped me understand the mindset of an incident responder. I learned how important accurate documentation is, how to recognize unusual activity, and how organizations detect and respond to threats. This project strengthened my ability to think critically and track security events in a structured way.
